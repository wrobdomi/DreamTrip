# Application features
  + Sign up and login
  + Display trips offer
  + Detail view of a trip
  + Trip ratings (if user bought the trip)
  + Adding and removing trips from basket
  + Buying trips
  + Shopping history
  + Adding comments to trip (if user bought the trip)
